Altere as datas do seu Wordpress para o Calendário Dekatrian.

== Description ==
>**“Não meçam o tempo com uma regua torta.”**
>*– PENA, Roberto*

[Dekatrian](http://dekatrian.com) é uma proposta criada pelo [Roberto "Pena" Spinelli](https://twitter.com/Peninha_13) como uma opção planejada ao caótico Calendário gregoriano.

Esse plugin permite alterar as datas do seu Wordpress para o Calendário Dekatrian.

== Installation ==
1. Faça o upload do plugin através do painel do WordPress;
2. Ative o plugin indo até o menu "Plugins";
3. Configure suas opções em "Configurações" > "Calendário Dekatrian".
